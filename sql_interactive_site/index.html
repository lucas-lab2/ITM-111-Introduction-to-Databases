What is a Database?

A database is an organized collection of data. In a relational database like MySQL, data is stored in tables.

A table is like a spreadsheet, with columns (which define the data types, like 'name' or 'age') and rows (which represent individual records).

Our example database is named `school` and has a table called `students`.

`SELECT` Command

The `SELECT` command is used to fetch data from a table.

SELECT * FROM students;
SELECT name, email FROM students;

`WHERE` Clause

The `WHERE` clause is used to filter records.

SELECT * FROM students WHERE age > 20;

Manipulating Data (DML)

`INSERT INTO`: Adds a new row.

INSERT INTO students (name, email, age) VALUES ('Carlos Lima', 'carlos@email.com', 23);

Creating Tables (DDL)

`CREATE TABLE` creates a new table.

CREATE TABLE courses (id INT, course_name VARCHAR(100));
